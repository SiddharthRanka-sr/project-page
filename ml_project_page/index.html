Abstract

Stroke is a leading cause of death and long-term disability worldwide. Early detection plays a vital role in minimizing its impact, but traditional diagnosis methods can be time-consuming and prone to human error. In this project, we propose a machine learning-based approach to predict the likelihood of stroke using health-related features such as age, hypertension, heart disease, glucose levels, BMI, and lifestyle habits.

The dataset used was publicly available and imbalanced, containing far fewer stroke cases than non-stroke cases. To overcome this, we applied SMOTE (Synthetic Minority Oversampling Technique) to balance the classes. After preprocessing the data — including handling missing values and encoding categorical variables — we trained and evaluated multiple machine learning models. The Random Forest classifier achieved the best results, offering high accuracy and balanced performance across precision and recall metrics.

Stroke Prediction Problem

Accurately predicting stroke risk is a key challenge in healthcare. The major difficulties include:

  • The class imbalance between stroke and non-stroke cases
  • The need to identify and analyze contributing health features
  • Ensuring models generalize well on unseen patient data
  • Supporting early intervention and decision-making

Models and Techniques Used

We experimented with multiple machine learning models including:

  • Random Forest
  • Support Vector Machine (SVM)
  • MLP
  • GMM

Key preprocessing steps included:

  • Handling missing values (especially in BMI)
  • Encoding categorical variables
  • Balancing the dataset using SMOTE

The Random Forest classifier achieved the highest performance with an accuracy of 0.9254%, and balanced precision, recall, and F1-score.

We also used visual tools such as:

  • Confusion Matrix: to evaluate predictions against actual labels
  • ROC Curve: to visualize model performance across thresholds
  • Feature Importance Plot: to interpret which features influenced the model most
